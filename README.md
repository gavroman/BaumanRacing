# Bauman Racing - многопользовательские гонки

## Зависимости
- [SFML](https://www.sfml-dev.org/download.php)
- [cmake](https://cmake.org/download/)
- [boost](https://www.boost.org/)

## Использование
#### Запуск серевера
* `cd server` 
* `cmake .`
* `make server.out`
* `./server.out`
#### Запуск клиента
* `cd ../client` 
* `cmake .`
* `make client.out`
* `./client.out`
